# Hexadata

Hexadata es un emprendimiento personal enfocado en el desarrollo de soluciones tecnológicas, productos educativos y experiencias digitales con un enfoque práctico, claro y útil para las personas y organizaciones que las adoptan.

Este repositorio contiene el sitio web de la marca, donde se comparten los proyectos, productos y la visión detrás de la propuesta.

## ¿Qué es Hexadata?

Hexadata nace como un espacio para combinar tecnología, educación y diseño de soluciones concretas. El objetivo es crear productos y contenidos que ayuden a resolver problemas reales mediante herramientas simples, eficientes y bien diseñadas.

## Proyectos y enfoque

Entre los principales ejes de trabajo se encuentran:

- Desarrollo de productos tecnológicos
- Soluciones educativas y experimentales
- Diseño de experiencias digitales
- Proyectos de hardware y software orientados a la práctica

## Tecnologías utilizadas

Este sitio está construido con:

- Zola como generador de sitios estáticos
- Markdown para el contenido
- Sass y plantillas personalizadas para el diseño
- Docker para facilitar la ejecución local

## Ejecutar localmente

Para ver el sitio en tu entorno local, puedes usar Docker Compose:

```bash
docker compose up --build
```

Luego abre tu navegador en:

```text
http://localhost:1111
```

## Estructura del proyecto

- site/: contenido del sitio y configuración de Zola
- themes/: tema visual del sitio
- public/: recursos estáticos
- docker-compose.yml: configuración para ejecutar el sitio localmente
